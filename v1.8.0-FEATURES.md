# AI Privacy Shield v1.8.0 - Feature Release

## 🎯 New Features

### 1. Per-Site Bypass (⏸️ Pause Button)
**What:** Temporarily disable blocking on specific sites for 10 minutes
**Why:** When something breaks, users can quickly test without uninstalling
**How to use:**
1. Click extension icon
2. Click "⏸️ Pause on This Site (10 min)"
3. Blocking resumes automatically after 10 minutes

### 2. Report New Tracker (🚨 Button)
**What:** One-click reporting of new surveillance endpoints
**Why:** Crowdsources detection, keeps rules updated
**How to use:**
1. Click extension icon
2. Click "🚨 Report New Tracker"
3. GitHub issue template opens pre-filled
4. Add details and submit

### 3. Diagnostics Panel (📊 Real-Time)
**What:** Shows what's being stripped/blocked in last 15 minutes
**Why:** Builds trust through visibility - users see the invisible work
**How to use:**
1. Click extension icon
2. Click "📊 Show Diagnostics"
3. See counts:
   - Headers stripped (x-client-data)
   - Client Hints blocked (sec-ch-ua-*)
   - Analytics cookies deleted
   - Surveillance endpoints blocked

### 4. Export Logs (💾 JSON Download)
**What:** Download blocking statistics as JSON
**Why:** Evidence collection, compliance reporting, personal records
**How to use:**
1. Click extension icon
2. Click "💾 Export Logs (JSON)"
3. File downloads: `ai-privacy-shield-logs-{timestamp}.json`
4. **No PII** - only statistics

### 5. Weekly Scorecard Generator
**What:** Automated report generation from exported logs
**Why:** Public accountability, marketing content, transparency
**How to use:**
```bash
# Export logs from extension first
node generate-scorecard.js ai-privacy-shield-logs-*.json > weekly-report.md
```
